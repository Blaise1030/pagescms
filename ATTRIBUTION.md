## What changed here

This fork explores a different deployment direction:

- Cloudflare Workers runtime via `vinext`
- Cloudflare D1 (SQLite) instead of PostgreSQL
- Self-hosted-first workflow for this repository

Features, docs, and behavior may diverge from upstream over time. For the official hosted product and canonical documentation, use [pagescms.org](https://pagescms.org).
